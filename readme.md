# Portal

Este projeto contém um portal para controle de pedidos e estoque, feito com HTML, CSS e JavaScript.
Nele cada representante tem acesso ao seu dashboard com seus indicadores de perfomance e
pode acompanhar sua carteira de clientes assim como o estoque atualizado em tempo real.

## Arquivos principais

- `index.html` - página de login ou entrada
- `home.html` - painel principal do portal
- `estoque.html` - gerenciamento de estoque
- `pedidos.html` - gerenciamento de pedidos
- `css/` - folhas de estilo
- `js/` - scripts e dados
- `img/` - imagens usadas pelo portal
